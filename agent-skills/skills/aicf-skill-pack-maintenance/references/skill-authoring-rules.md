# Skill Authoring Rules

- Folder name and frontmatter `name` must match.
- Use lowercase kebab-case and avoid consecutive hyphens.
- License must be MIT for this package.
- Include compatibility and required AICF metadata keys.
- Keep `SKILL.md` under 500 lines.
- Include required body sections in validator order.
- Link only one-level `references/`, `assets/`, or `scripts/` files.
- Keep examples synthetic and public-safe.
