# Skills

This directory is the source of truth for AICF Agent Skills.

It contains the complete 17-skill v1 inventory for AICF builder workflows:
capability authoring, governance, policy/risk, evals, security, runtime,
action lifecycle, trust/redaction/retention, observability/replay, controls,
provider conformance, optional storage/AWS guidance, migration, docs/examples,
control-plane UI/API guidance, release hygiene, and skill-pack maintenance.

Do not mirror these folders into local discovery directories by default. Use
the package install script when a local copy is needed.
