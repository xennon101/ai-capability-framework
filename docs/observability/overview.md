# Observability Overview

AICF observability emits sanitized metadata events and spans. It does not store raw
prompts, raw provider payloads, traces, secrets, or sensitive tool output by default.

Read:

- [Observability runtime](../observability-runtime.md)
- [Live evals](../live-evals.md)
- [EvalOps export interfaces](../evalops.md)

OpenTelemetry and Langfuse integrations remain optional and isolated.
