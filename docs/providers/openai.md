# OpenAI

AICF supports OpenAI in two ways:

- descriptor export from Core for OpenAI Responses tools;
- an optional OpenAI Responses runtime loop under `ai-capability-framework/openai`.

Start here:

- [OpenAI walkthrough](../openai-walkthrough.md)
- [OpenAI Responses descriptor adapter](../openai-responses.md)
- [OpenAI Responses runtime](../openai-runtime.md)

Default examples do not call OpenAI. Live use requires a caller-provided client
or optional SDK setup.
