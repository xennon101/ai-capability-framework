# Control Plane Overview

The AICF control plane is a self-hostable local/reference review surface for
capabilities, governance status, eval/security/conformance coverage, approvals,
controls, replay metadata, and evidence exports.

Read:

- [Control plane](../control-plane.md)
- [Governance gate](../governance/gate.md)
- [Evidence export](../evidence.md)

The reference app does not provide production auth, production persistence, hosted SaaS
behavior, model calls, or side-effect execution.
